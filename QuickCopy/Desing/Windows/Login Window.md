This is a very basic window whit a clear goal and a simple way of doing it. 
There are 4 points of interest :
- 2 buttons
	- Login button
	- Sign in button
- 2 input fields
	- Username
	- Password

# Login button

Checks of the user is in the data base. And checks after that that the given password is the same as the password in the database for this user.
If the password is not given a error notice to the user.
If the password is incorrect give a error notice to the user.
Only if both username and the password are right and validated by the database will the user be allowed to enter the application. And the [[Copy Window]] will be opened up.

# Sign in button 

When this button is pressed the [[Sign in Window]] will be opened up.
